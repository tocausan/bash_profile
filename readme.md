# .bash_profile #

## Description ##
Update your Terminal profile template

## How to  ##
```bash
# clone this repo
git clone <this repo> <optional given name>
# enter the source
cd bash_profile
# copy the content to your user profile
sudo cp bash_profile ~/.bash_profile
# update the file if wanted
sudo nvim ~/.bash_profile
```
